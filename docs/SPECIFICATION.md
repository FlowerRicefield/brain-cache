# Brain Cache Clear - Ultimate Pro 仕様書

## 1. 概要

**アプリケーション名:** Brain Cache Clear - Ultimate Pro v1.1

**目的:** ユーザーの頭の中にある未処理の思考（タスク、アイデア、感情など）を素早く記録・整理・管理するウェブアプリケーション

**対象ユーザー:** GTDやタスク管理を実践したい個人ユーザー、思考整理が必要な知識ワーカー

**言語:** 日本語対応

---

## 2. コア機能一覧

### 2.1 脳内キャッシュの記録・解放（Main Input）

| 機能 | 説明 | 詳細 |
|------|------|------|
| **テキスト入力** | メインのテキストエリアに思考を入力 | 複数行対応、改行可能 |
| **タイマー計測** | 入力開始から経過時間を自動計測 | 入力時点でタイマー開始、0秒～制限なし |
| **カテゴリ自動判定** | 入力テキストから自動的にカテゴリを推測 | リアルタイム判定、Visual Indicator付き |
| **脳内解放（投稿）** | 入力内容をキャッシュとして保存 | Enter キー or ボタンクリックで投稿 |

### 2.2 カテゴリ分類

以下4つのカテゴリ自動判定ロジック:

| カテゴリ | アイコン | 判定キーワード | 色 |
|---------|---------|--------------|-----|
| **Inbox** | 📥 | デフォルト | slate-gray |
| **Task** | ✓ | todo, やる, タスク, `- ` 記法 | orange |
| **Idea** | 💡 | アイデア, ひらめき, 開発 | cyan |
| **Emotion** | ❤️ | うれしい, つらい, むかつく | pink |

### 2.3 キャッシュアイテムの表示・管理

| 機能 | 説明 | UI位置 |
|------|------|--------|
| **一覧表示** | キャッシュをカード形式で表示 | メインコンテナ |
| **カテゴリフィルタ** | カテゴリチップをクリックして絞り込み | ヘッダー下・チップエリア |
| **深掘りスレッド** | 1つのキャッシュに関連メモを追加 | サイドドロワー |
| **アーカイブ** | 完了アイテムを別ビューへ移動 | タブ切替 |
| **完全削除** | キャッシュを永遠に消去 | カードのボタン |

### 2.4 タブビュー

| タブ | 表示内容 | 対象 |
|-----|---------|------|
| **Today** | 未処理のアクティブキャッシュ一覧 | `isArchived: false` |
| **Archive** | 処理済み/アーカイブ済みアイテム | `isArchived: true` |

### 2.5 深掘りスレッド機能（Thread Drawer）

| 要素 | 説明 |
|-----|------|
| **スレッドオープン** | キャッシュカードの「深掘りボタン」をクリック |
| **タイマー** | 5分のカウントダウン制限 |
| **メッセージ送信** | テキスト入力 → Enter で追加メモを記録 |
| **メタ情報** | 元となったキャッシュ内容と作成時刻を表示 |
| **クローズ** | ドロワー右上の X ボタン or 動作完了後 |

### 2.6 サルベージ・デック機能（Salvage Deck）

| 機能 | 説明 | トリガー |
|-----|------|---------|
| **デック準備** | 最大5件の古いアクティブキャッシュを抽出 | 「サルベージ」ボタン |
| **カード表示** | Tinder風のマッチングUI | デッキスロット内 |
| **判定ジェスチャ** | 左右スワイプ or ボタンで判定 | PointerEvents |
| **アクション** | Keep(保持) / Archive / Delete | 各ボタン or スワイプ |
| **セッション終了** | 全デック完了表示 → 「完了して戻る」で Today タブへ |  |

## 3. データモデル

### 3.1 Cache アイテム

```json
{
  "id": "auto-generated-id",
  "content": "ユーザー入力のテキスト",
  "category": "inbox|task|idea|emotion",
  "isArchived": false,
  "elapsedSeconds": 45,
  "createdAt": "2026-06-06T12:34:56Z",
  "threads": [
    {
      "text": "深掘りメモ1",
      "createdAt": "2026-06-06T12:35:00Z"
    }
  ]
}
```

### 3.2 Firestore スキーマ

```
/users/{uid}/
  /caches/{docId}
    - content (string)
    - category (string)
    - isArchived (boolean)
    - elapsedSeconds (number)
    - createdAt (timestamp)
    - threads (array of objects)
```

### 3.3 ローカルストレージ フォールバック

- **キー:** `brain_cache_mock`
- **値:** JSON 文字列の配列（全 Cache アイテム）
- **用途:** Firebaseが利用不可時のモード

## 4. UI / UX 仕様

### 4.1 ヘッダー

```
[ロゴ + タイトル] ← 固定
       [Google連携ボタン] [サルベージボタン] [同期ステータス]
```

| 要素 | 説明 |
|-----|------|
| **ロゴアイコン** | 稲妻アイコン + グラデーション背景 |
| **タイトル** | "BRAIN CACHE CLEAR / Ultimate Pro v1.1" |
| **Google連携** | 未連携時: 「Google連携」/ 連携済み時: 「連携完了」(緑) |
| **サルベージボタン** | グラデーション紫、常に活動可 |
| **同期ステータス** | ☁️クラウド(正常) / ⚠️警告(エラー) / 📡オフライン(ローカル) |

### 4.2 メインコンテンツエリア

#### 入力セクション
- テキストエリア 3行
- タイマー表示（MM:SS）
- プログレスバー（0～60秒）
- カテゴリ自動判定表示（💡タスク自動判定）
- 「脳内解放」ボタン

#### フィルタエリア
- タブ: Today / Archive
- カテゴリチップ: Inbox / Task / Idea / Emotion
- ドラッグ可能（カテゴリ変更用）

#### キャッシュ一覧
- カード形式（border-left のカラーインジケータ）
- 各カード内容:
  - テキスト（複数行対応、改行表示）
  - カテゴリバッジ
  - 経過時間（分:秒）
  - スレッド数表示
  - アクションボタン（深掘り / アーカイブ / 削除）

## 5. ユーザーフロー

### 5.1 基本的な使用フロー

```
1. ページ起動
   ↓
2. Firebaseまたはローカル初期化
   ↓
3. ユーザー入力（テキストエリア）
   ↓
4. タイマー開始 + カテゴリ自動判定
   ↓
5. 「脳内解放」で投稿
   ↓
6. キャッシュ一覧にカード追加
   ↓
7. 分類・深掘り・アーカイブ / 削除
```

### 5.2 深掘りフロー

```
カードの「深掘り」ボタン
       ↓
スレッドドロワー オープン
       ↓
5分タイマー開始
       ↓
メモ追加（Enter送信）
       ↓
完了 → ドロワー×で閉じる
```

### 5.3 サルベージフロー

```
「サルベージ」ボタン
       ↓
古いキャッシュ5件を抽出
       ↓
Tinder風マッチングUI
       ↓
左スワイプ/右スワイプ or ボタン判定
       ↓
アクション実行
       ↓
次のカード表示
       ↓
全完了 → 「完了して戻る」
```

## 7. 認証 & 同期

### 7.1 認証フロー

| ステップ | 処理 | 状態 |
|---------|------|------|
| **初回起動** | `signInAnonymously()` | 匿名認証 |
| **Google連携** | `GoogleAuthProvider` + `signInWithPopup()` | プロバイダ統合 |
| **連携確認** | `providerData` チェック | UI更新 |

### 7.2 リアルタイム同期

- **リスナー:** `onSnapshot()` で Firestore を監視
- **トリガー:** CRUD 操作時に自動更新
- **エラーハンドリ:** 接続失敗時 → ローカル保存モードへ自動フォールバック

### 7.3 同期ステータス表示

| ステータス | アイコン | 色 | 意味 |
|----------|---------|-----|------|
| **同期完了** | ☁️ | emerald | Firestore正常 |
| **同期エラー** | ⚠️ | orange | 接続エラー中 |
| **オフライン** | 📡 | slate | localStorage のみ |

## 8. キーボード & ジェスチャ

### 8.1 キーボード操作

| キー | 動作 |
|-----|------|
| **Enter** (メイン入力) | テキスト送信 |
| **Ctrl+Enter** | 改行のみ |

### 8.2 ジェスチャ

| ジェスチャ | 動作 | 対象 |

## 9. パフォーマンス & 制限

| 項目 | 値 | 説明 |
|-----|-----|------|
| **タイマー上限表示** | 60秒 | 超過時は% で表示 |
| **スレッド制限時間** | 5分(300秒) | カウントダウン |
| **サルベージデック** | 最大5件 | 古い順 |
| **メッセージログスクロール** | 自動 scroll-to-bottom | 新規追加時 |

---

## 10. エラーハンドリング

| シナリオ | 対応 |
|---------|------|
| **Firebaseが利用不可** | localStorage へフォールバック |
| **ネットワーク切断** | 同期ステータスを警告表示 |
| **入力が空の状態で送信** | 処理スキップ（トースト非表示） |
| **サルベージ対象がない** | トースト: 「現在サルベージ可能なアクティブキャッシュがありません」 |

## 14. まとめ

**Brain Cache Clear - Ultimate Pro** は、ユーザーの思考を素早く記録し、カテゴリ分類・深掘り・サルベージ機能を通じて**効率的なタスク管理と思考整理**を実現するWebアプリケーションです。Firestore による リアルタイム同期と匿名認証により、複数デバイスでのシームレスな体験を提供します。
